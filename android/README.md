# MediaPlayer Android 客户端

专为 Android TV、网络机顶盒以及 Android 智能设备打造的专业级流媒体播放客户端。

## 🛠️ 编译与开发环境

- **IDE**: Android Studio Hedgehog (2023.1.1) 或更高版本
- **JDK 版本**: Java 17
- **Compile SDK**: 35
- **Min SDK**: 23 (Android 6.0+)
- **Target SDK**: 34

---

## 🚀 快速开始

### 1. 克隆代码
```bash
git clone https://github.com/kuai410022283/mediaplayer.git
cd mediaplayer/android
```

### 2. 编译 Release APK
```bash
# Windows
.\gradlew.bat assembleRelease

# Linux / macOS
./gradlew assembleRelease
```
编译产物目录位于：`app/build/outputs/apk/release/`
