# 开源贡献指南 (Contributing Guide)

感谢您对 **MediaPlayer** 项目感兴趣！我们欢迎各种形式的贡献，包括修复 Bug、改进文档、提交新功能或提出建议。

---

## 🛠️ 提交贡献流程

1. **Fork 本仓库** 到您自己的 GitHub 账号。
2. **创建特性分支**：
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. **提交代码变更**：
   确保您的代码符合规范，注释清晰。
4. **Push 到您的分支**：
   ```bash
   git push origin feature/amazing-feature
   ```
5. **提交 Pull Request (PR)**：
   在 GitHub 上发起 Pull Request，简要说明您的修改内容和测试结果。

---

## 🐛 报告问题 (Issue)

如果您发现了 BUG 或有改进建议，请按以下要求提交 Issue：
- 描述清晰的问题现象与复现步骤。
- 附带必要的设备型号（如 Android TV 品牌、芯片）和日志（Logcat）。

---

## 📜 行为准则

请保持友好交流与相互尊重，共同维护良好的开源社区氛围。
